# AI Resume Analyzer

## 📌 Project Overview
AI Resume Analyzer is a smart web-based application that analyzes a candidate's resume and compares it with a Job Description (JD). It helps users understand how well their resume matches the job requirements using ATS-style scoring.

## 🎯 Objective
To help students and job seekers improve their resumes and increase chances of selection in recruitment processes.

## 🚀 Features
- Upload Resume (PDF/DOCX)
- Paste Job Description
- Resume vs JD Match Score
- Missing Keywords Detection
- Skills Extraction
- Suggestions to Improve Resume
- User Friendly Interface

## 🛠️ Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- NLP (Natural Language Processing)
- Machine Learning
