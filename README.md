# HireLens AI 🚀

## AI Resume – JD ATS Match Analyzer with Login System

HireLens AI is an AI-powered web application that compares a candidate’s **Resume** with a **Job Description (JD)** and generates an ATS-style match report.

The system now includes **User Registration and Login**, making it a more professional and real-world recruitment support platform.

---

## ✨ Features

✅ User Registration
✅ User Login Authentication
✅ Upload Resume (PDF)
✅ Paste Job Description
✅ ATS Match Score (0–100)
✅ Rating Label (Excellent / Good / Average / Needs Improvement)
✅ Matched Skills Detection
✅ Missing Skills Detection
✅ Missing Keywords Detection
✅ Resume Improvement Suggestions
✅ ATS Optimization Tips
✅ Clean Professional UI

---

## 🧠 How It Works

1. User registers a new account
2. User logs into the system
3. Uploads resume PDF
4. Pastes Job Description
5. AI analyzes resume vs JD
6. Generates ATS score and recommendations
7. Displays result dashboard
